# 动效规范

## 结论
动效只服务反馈、层级和陪伴感，不做炫技。核心动效是页面淡入、卡片轻反馈、活动筛选过渡、签到盖章和日期点亮。

## Token
- `--motion-fast`: 150ms。
- `--motion-base`: 220ms。
- `--motion-slow`: 360ms。
- `--ease-standard`: `cubic-bezier(0.22, 1, 0.36, 1)`。
- `--ease-gentle`: `cubic-bezier(0.2, 0, 0, 1)`。

## 使用场景
- 页面进入：透明度 0 到 1，Y 轴 10px 到 0。
- 卡片悬停：Y 轴 -2px，阴影轻微增强。
- 筛选变化：列表内容 crossfade。
- 签到：印章缩放 0.94 到 1，透明度进入。
- 数字：不做夸张翻牌，使用轻微淡入。

## 减少动态效果
当 `prefers-reduced-motion: reduce`：
- 关闭持续摆动和滚动进入。
- 页面切换缩短到 80ms 或直接显示。
- 保留按钮 pressed/focus 状态。

## 禁止
烟花、金币飞出、宝箱、抽奖机、转盘、夸张弹跳、大幅旋转、强视差。
