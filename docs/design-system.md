# 设计系统

## 结论

当前视觉方向为“真实劳作纪实 + 田野成长档案”。首页首屏不再把十位成员群像作为主视觉，而是以原创麦田、土地、天空和乡村环境图建立自然劳作氛围；十位成员入口下沉为“十人田野档案”。最终效果应像一本持续更新的田野日志，而不是普通粉丝站、游戏社区或娱乐资讯页。

## 设计原则

- 自然但不土气：用低饱和田野色、纸张纹理和真实环境图，不堆廉价装饰。
- 温暖但不甜腻：保留米白纸张和麦穗金，但主按钮与导航以深麦田绿控制气质。
- 青年但不幼稚：字体、留白和网格偏纪实杂志，不使用可爱手账或游戏化效果。
- 有粉丝氛围但不饭圈化：成员页是“档案”，社区是“田埂留言”，动态是“后陡门通讯”。
- 有手作感但专业：纹理透明度低，所有交互状态、表单和响应式布局保持清晰。

## 颜色 Token

已落实到 `src/index.css` 的 Tailwind v4 `@theme`：

- `--color-field-green`: `#244D38`，主按钮、导航强调、成功与自然感。
- `--color-sprout-green`: `#6F8B58`，辅助模块、签到成长阶段。
- `--color-wheat-gold`: `#D4A94F`，活动、重点数字和麦穗强调。
- `--color-soil-brown`: `#765A3B`，土地感辅助文字和部分强调。
- `--color-paper-white`: `#F4EFE3`，页面主背景。
- `--color-sky-blue`: `#C8DCE5`，待确认、加载、轻提示。
- `--color-charcoal`: `#2B2D29`，正文主文字。
- `--color-muted-text`: `#73766D`，辅助文字。
- `--color-border-color`: `rgba(74, 73, 61, 0.16)`，纸张边线。

兼容旧类名的别名仍保留：`field-bg`、`field-surface`、`field-muted`、`field-ink`、`field-soft`、`wheat`、`olive`、`sky-wash`、`brick`、`paper-line`。

## 字体

- 正文：系统优先的中文无衬线字体，依次使用 `PingFang SC` / `Microsoft YaHei UI` / `Microsoft YaHei` / `Noto Sans SC`，保证 Windows、macOS 和移动端阅读稳定。
- 标题：沿用同一中文无衬线体系，通过 `600-700` 字重、字号和留白建立层级；不在栏目间混用楷体、宋体或书法字体。
- 数字和英文：`Segoe UI` / `Arial`，配合 `tabular-nums` 用于日期、签到数和活动时间，不让中文标签落入代码等宽字体。
- 装饰性字体只允许出现在极短的英文眉题或素材图内，不能用于正文、筛选器、卡片标题和连续阅读内容。
- 旧有 `font-serif`、`font-mono` 工具类作为兼容别名，分别映射到标题字体和数据字体；新增页面优先使用语义化字体 Token。

## 材质

- 页面背景：米白纸张底，叠加 2%-5% 纸张颗粒和低透明网格。
- 卡片：纸张层次、1px 低透明边框、极轻阴影。
- 标签：小矩形纸签，减少胶囊形。
- 签名元素：`field-tag`、`date-spine`、`journal-rule`、`field-note`。

## 组件规则

- 主按钮：深麦田绿底、米白文字，hover 仅轻微加深和上移 1px。
- 次按钮：透明纸张底、绿色细边框。
- 卡片：12px-18px 圆角，轻阴影，避免所有内容都被厚重卡片包裹。
- 状态标签：低饱和绿色、麦金、天空蓝和砖红，不使用荧光色。
- 输入框：米白纸张底、细边框、清晰焦点。

## 页面应用

- 首页 Hero：原创田野环境 WebP，包含日期、天气、田野记录信息；十位成员不在首屏堆头像。
- 成员：十人田野档案，含编号、标签、占位说明和授权素材提示。
- 活动：田野月历与移动端时间线，个人/多人/团体活动后续可继续细化状态色。
- 签到：种子、发芽、幼苗、抽穗、麦田五阶段，保留盖章感，不做金币动画。
- 动态：后陡门通讯 / 今日田野记录，突出日期、来源和人物关联。
- 社区：田埂留言 / 麦田回声，留言卡米白纸张和深绿色文字。

## 动效

- 页面和图片进入：300ms-700ms 淡入、轻微上移。
- 卡片 hover：最多上移 1-2px，图片最多放大 1.02。
- Hero 图片：轻微模糊到清晰，不做人像飞入或强视差。
- reduced motion：关闭持续或复杂动画，仅保留必要状态反馈。

## 媒体

- 首页主视觉：`src/assets/images/field-hero-documentary.webp`，原创生成田野环境图，不含真人脸、Logo 或官方海报元素。
- 成员真实照片、群像、视频上线前必须确认授权。
- 所有图片需要 alt、来源字段和加载失败占位。
